## Próximos Passos Imediatos

### HOJE (2006-06-18)

1. **Instalar Dashboard (.240)** ⭐
   ```bash
   cd /root/.openclaw/workspace/docs/dashboard
   docker network create omega-net
   docker-compose up -d
   ```

2. **Instalar HedgeDoc (.241)**
   ```bash
   docker run -d --name hedgedoc \
     --restart unless-stopped \
     -p 3001:3001 \
     -v /home/ceops/data/hedgedoc:/hedgedoc/public/user-files \
     quay.io/hedgedoc/hedgedoc:latest
   ```

3. **Instalar code-server (.241)**
   ```bash
   docker run -d --name code-server \
     --restart unless-stopped \
     -p 8080:8080 \
     -v /home/ceops/data/code-server:/home/coder/.local/share/code-server \
     -v /home/ceops/workspace:/workspace \
     -e PASSWORD=*** \
     codercom/code-server:latest
   ```
